Sobre o Curso

O curso de Desenvolvimento de Software Multiplataforma da Fatec Jaú capacita os alunos a desenvolver aplicações que funcionam em diferentes ambientes, garantindo versatilidade e eficiência nas soluções propostas.

Matriz Curricular

  • Programação Orientada a Objetos
  • Desenvolvimento Web
  • Aplicações Mobile
  • Bancos de Dados
  • Engenharia de Software
  • Testes de Software
  • Interfaces de Usuário
  • Desenvolvimento de APIs
